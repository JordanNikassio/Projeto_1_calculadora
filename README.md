# Projeto_1_calculadora
Primeiro projeto em git do curso analista de dados Ebac

## Nesse projeto está uma calculadora simples de média para estudantes

## Executando o arquivo .sh
 No terminal WSL apenas digite ./calculadora.sh

## Arquivo código python
No início da execução do código o usuário deve fornecer o nome do aluno a ser avaliado.
em seguida deve fornecer as médias anteriores das quatro unidades e, por fim, o código executa o calculo da média final.
Se a média do aluno for igual ou superior a 7.0, o programa deve retornar uma mensagem de 'Aluno aprovado', caso contrário
deve retornar 'Aluno reprovado'.
